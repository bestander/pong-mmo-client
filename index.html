API

This web project has the following setup:

  • component.json - component library descriptor
  • Makefile - make descriptor for downloading resources, building and cleaning a build. Requires component(1) and uglifyjs to be installed
  • index.js - main CommonJS module that binds all the pieces together
  • src/ - all sources are here
  • tests/ - jasmine unit tests and test spec runner
  • Makefile - standard compilation descriptor for component
  • demo/ - folder for example files
  • build/ - folder that is created by component with make build command
  • components/ - folder with project dependencies, will be created by component with 'make components' command
  • MIT-LICENSE - license wording
